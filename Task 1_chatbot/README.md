
# 🤖 AI Chatbot

An intelligent chatbot built using **Python** that interacts with users through natural language conversations. The chatbot processes user input, identifies common queries, and provides relevant responses in a simple and interactive command-line interface.

## 📌 Features

* Interactive text-based conversation
* Responds to greetings and common questions
* Simple keyword-based response system
* Easy to customize with new responses
* Beginner-friendly Python implementation
* Clean and modular code structure

## 🛠️ Technologies Used

* Python 3
* Standard Python Libraries

## 📂 Project Structure

```
AI-Chatbot/
│── chatbot.py
│── README.md
```

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder in VS Code or any Python IDE.
3. Run the chatbot using:

   ```
   python chatbot.py
   ```
4. Start chatting with the bot.
5. Type **exit** or **quit** to end the conversation.

## 💡 Sample Conversation

```
You: Hello
Bot: Hi! How can I help you today?

You: What is Python?
Bot: Python is a popular programming language used for web development, AI, automation, data science, and more.

You: Thank you
Bot: You're welcome! Have a great day.

You: exit
Bot: Goodbye! Have a nice day.
```
