# ❌⭕ Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe** game developed using **Python**. This is a command-line application where two players take turns marking spaces on a 3×3 grid. The game automatically detects wins, draws, and validates user input to ensure a smooth gaming experience.

## 📌 Features

* Two-player gameplay
* Interactive command-line interface
* Displays the game board after every move
* Detects winning combinations
* Identifies draw situations
* Prevents invalid or occupied moves
* Option to play multiple rounds (if implemented)

## 🛠️ Technologies Used

* Python 3
* Standard Python Libraries

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder in VS Code or any Python IDE.
3. Run the program using:

   ```
   python tic_tac_toe.py
   ```
4. Follow the on-screen instructions to play.
5. The game ends when a player wins or the match results in a draw.

## 🎮 How to Play

* The game is played on a **3×3 grid**.
* **Player X** always starts first.
* Players take turns entering the position where they want to place their symbol.
* The first player to align **three symbols horizontally, vertically, or diagonally** wins.
* If all spaces are filled without a winner, the game ends in a draw.
