# 🎵 Music Recommendation System

A simple **Music Recommendation System** built using **Python** and **Machine Learning** that suggests songs based on user preferences. The system analyzes similarities between songs and recommends tracks with comparable characteristics, helping users discover new music they may enjoy.

## 📌 Features

* Recommends similar songs based on user selection
* Uses machine learning techniques for recommendation
* Fast and efficient recommendation process
* Easy-to-use command-line interface
* Simple and modular code structure
* Beginner-friendly implementation

## 🛠️ Technologies Used

* Python 3
* Pandas
* NumPy
* Scikit-learn


## ▶️ How to Run

1. Clone this repository.
2. Open the project folder in VS Code or any Python IDE.
3. Install the required libraries:

   ```
   pip install pandas numpy scikit-learn
   ```
4. Run the program:

   ```
   python recommendation_system.py
   ```
5. Enter a song name when prompted to receive similar music recommendations.

## 🎵 How It Works

* Loads the music dataset.
* Processes song information using machine learning techniques.
* Calculates similarities between songs.
* Matches the user's selected song with the dataset.
* Displays a list of recommended songs based on similarity scores.
